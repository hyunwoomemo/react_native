# React Native 마스터클래스

## 프로젝트 시작하기

### react-native init

```bash
npm react-native init
```

나쁘지 않지만 너무 기본만 주어짐

### ignite

react native CLI로 초기 셋팅이 훌륭함
(navigation, typescript, asyncstorage 등등)

```bash
npx ignite-cli new 서비스명
```

### create react native app

페이스북 + expo 에서 만든 것

ios, android 파일에 접근 가능하면서 expo도 사용할 수 있음
